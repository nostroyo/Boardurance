This folder contains system architecture documentation, design patterns, and structural decisions.

## Contents

## Architecture Patterns

### Backend Architecture
- **Domain-Driven Design** following "Zero to Production in Rust"
- **Hexagonal Architecture** with ports and adapters
- **Clean separation**: Domain → Routes → Infrastructure

### Frontend Architecture
- **Component-based** React architecture
- **Context providers** for state management
- **Service layer** for API communication

### Blockchain Architecture
- **Anchor Framework** for smart contracts
- **Metaplex standards** for NFT compatibility
- **Web3 wallet integration**
