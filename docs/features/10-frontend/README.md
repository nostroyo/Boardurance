# Frontend Documentation

This folder contains documentation specific to the React frontend application.

## Contents

- `FRONTEND_README.md` - React frontend documentation
- `UI_IMPROVEMENTS.md` - Interface enhancements and design

## Technology Stack

- **React 19.1.1** with TypeScript
- **Vite 7.1.2** for fast builds
- **Tailwind CSS 3.4.17** for styling
- **Solana Wallet Adapter** for Web3

## Key Features

- Game interface components
- Wallet integration
- NFT management UI
- Real-time race updates
- Player dashboard

## Development

```bash
cd empty-project
npm run dev          # Development server (port 5173)
npm run build        # Production build
```
