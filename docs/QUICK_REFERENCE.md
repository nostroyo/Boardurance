## üí° Tips

### Navigation
- Start with feature folder READMEs for overview
- Use cross-references to jump between related features
- Check implementation/ subfolders for detailed technical docs

### Search
```powershell
# Search all documentation
cd docs
Select-String -Path "features/**/*.md" -Pattern "your search term"

# Search specific feature
cd docs/features/02-racing-system
Select-String -Path "*.md" -Pattern "your search term"
```

### Updates
- When adding new docs, place in appropriate feature folder
- Update feature README when adding new files
- Add cross-references to related features
