# Product Overview

This workspace contains a **Web3 Game** project with NFT integration on Solana blockchain:

## Web3 Game Architecture
- **Frontend**: React + TypeScript + Vite web application for game interface
- **Smart Contract**: Solana NFT smart contracts using Anchor framework
- **Backend**: Rust-based backend services (to be implemented)

## Empty Project (Game Frontend)
React + TypeScript + Vite application that will serve as the game's user interface with:
- Modern React development with TypeScript
- Tailwind CSS for styling
- Web3 wallet integration capabilities
- NFT display and interaction features
- ESLint + Prettier for code quality
- Vite for fast development and building

## Solana Smart Contract (NFT System)
Anchor-based Solana smart contract project for NFT functionality including:
- NFT minting and management
- Game asset tokenization
- Player ownership verification
- Rust-based smart contract development using Anchor framework
- Test infrastructure for contract validation
- Deployment configuration for various Solana networks

## Task Management
- All development tasks are managed via **Dart AI using MCP**
- Tasks must be moved through states: **To Do → Doing → Complete**
- Task descriptions must be updated when moving between states
- No new subtasks should be created on the Dartboard
